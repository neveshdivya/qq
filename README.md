# Medicare UI

A React and Tailwind CSS v4 based UI for a Medicare application.

## Getting Started

1. Navigate to the project directory:
```bash
cd d:/qq/medicare-ui
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```
